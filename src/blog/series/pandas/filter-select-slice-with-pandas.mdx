## ðŸ§½ Filtering rows by condition

Letâ€™s say you want all products that cost more than 5:

---

## ðŸ§  Using `.loc[]` and `.iloc[]`

`.loc[]` is label-based.  
`.iloc[]` is index-based.

**Example with `.loc[]`:**

```python
# First row, all columns
df.loc[0]

# First 3 rows, specific columns
df.loc[0:2, ["product_name", "price"]]
```

**Example with `.iloc[]`:**

```python
# First row
df.iloc[0]

# First 5 rows and first 3 columns
df.iloc[0:5, 0:3]
```

## ðŸ§  Pro tip: Use `.query()` for readable filters


Much easier to read and chain when things get complex.
