## ğŸ”¢ Using `groupby()` + aggregation

Letâ€™s start simple: whatâ€™s the **average price per category**?


You can also get multiple aggregations:


## ğŸ§  Custom aggregations with `agg()`

You can even pass custom functions or lambdas:

```python

    average="mean",
    highest="max",
    products="count"
)
```

You can rename the output columns with `named aggregation` like above (`average=...`).

## ğŸ”„ Using `pivot_table()`

Pivot tables give you more flexibility for reshaping data. Letâ€™s say we want the **average price per category and availability**:


This shows how price changes depending on whether items are in stock or not.

## ğŸ” Reshaping with `.unstack()` and `.melt()`

You can convert groupby results into wide format with `.unstack()`:

```python

```

Or go from wide â†’ long format using `.melt()`:

